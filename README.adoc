= https://github.com/arkime/arkime/[Arkime] Docker image/container

This repository builds Arkime container on PhotonOS for CI/CD.


== Getting Started

=== Prerequisites

The build host (or cloud instance) requires:

- `docker`
- `make`


=== Build the container

----
% make arkime
----
